Windows 10/11

SecureUxThemePatcher - You need this to apply the following themes
  • Aero10 - Windows 7/Vista theme for Windows 10, put in "C:/Windows/Resources/Themes", you need a deviantart account to download this, DON'T USE THE SETTINGS APP TO APPLY!

  • Aero7-11 - Windows 7 theme for Windows 11, put in "C:/Windows/Resources/Themes", read the main page for the theme on Github, DON'T USE THE SETTINGS APP TO APPLY!

  • Luna10 - Windows XP theme for Windows 10/11, put in "C:/Windows/Resources/Themes", you need a deviantart account to download this, DON'T USE THE SETTINGS APP TO APPLY, REQUIRES BASICTHEMER2!

  • Aero10: 7000 - Windows 7 Beta theme for Windows 10/11, put in "C:/Windows/Resources/Themes", you need a deviantart account to download this, DON'T USE THE SETTINGS APP TO APPLY!

  • Longhorn Pearl - Windows Longhorn theme for Windows 10/11, put in "C:/Windows/Resources/Themes", you need a deviantart account to download this, DON'T USE THE SETTINGS APP TO APPLY!

  • Aero10: Metro - Windows 8.1 theme for Windows 10/11, put in "C:/Windows/Resources/Themes", you need a deviantart account to download this, DON'T USE THE SETTINGS APP TO APPLY!
