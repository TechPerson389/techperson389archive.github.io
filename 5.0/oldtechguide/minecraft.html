Guide to Java for Minecraft

Minecraft 1.11.2 and before = Java 6 (If you can, use Java 8 instead)

Minecraft 1.16.5 and before = Java 8

Minecraft 1.17 to 1.20.4 = Java 17

Minecraft 1.20.5 and after = Java 21
